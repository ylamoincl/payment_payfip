Module de paiement Tipi Régie

Description

Tipi Régie est un système de paiement en ligne proposé pa la Direction générale des Finances Publiques. Son but est de faciliter le paiement des services publics locaux.

Ce module de paiement permet de payer les commandes et factures, générées par Odoo, via le système de paiement Tipi Régie.

Installation

Afin de l'installer, il suffit d'aller dans la liste des applications et de cliquer sur "Installer"

Une fois l'installation terminée, le module peut être retrouvé dans la liste des intermédiaires de paiement dans l'application "Comptabilité".

La configuration peut ensuite être faite en trois étapes :

  • Ajout du numéro client correspondant au numéro d'identification de la collectivité chez Tipi.
  • Configurer la validation du workflow (validation ou non des commandes, factures, ...
  • Publication du mode de paiement si le module eCommerce est installé.

Usage

Le module peut être utilisé avec la boutique d'Odoo. Une fois le panier créé, le choix du module de paiement Tipi Régie est disponible.

Ensuite, un processus de paiement standard est à suivre sur le site gouvernemental de la DGFIP.

Une fois le paiement validé, vous pouvez revenir sur le site de la boutique avec le bouton "Retour site".
