# Payfip payment acquirer

## Install Odoo 17

- [Install Odoo 17.0](https://www.odoo.com/documentation/17.0/administration/on_premise/packages.html) on your favorite environment

## Install addon

- Execute command :

```ruby
pip install "git+https://github.com/Horanet/payment_payfip.git@17.0#egg=odoo17-addon-payment-payfip&subdirectory=setup/payment_payfip"
```
